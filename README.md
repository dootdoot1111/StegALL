> A MacOS tool that makes use of all available steganography tools very easy !

### 🏠 [Homepage](https://github.com/dootdoot1111/StegALL)

## Install

```sh
git clone https://github.com/dootdoot1111/StegALL.git
cd StegALL
```

## Usage

```sh
./stegall <file> |[-h] (for help)
```
